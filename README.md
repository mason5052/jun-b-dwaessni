# Jun B 됬니?

**"Jun B 됬니?"** - 발음: 준비됐니?

Jun (Junkuk) 과 B (Bilqees), 두 사람의 이름이 합쳐져 탄생한 제목.
"부부의 여정을 시작할 준비가 됐니?" 라는 물음이자, 두 사람이 하나가 되어가는 이야기.

---

## 이 책에 대하여

미국 동부, 조지아 중부의 작은 시골 마을.
한국에서 태어나 미국 땅에 뿌리를 내린 한 남자와, 그의 단 하나뿐인 파트너.

행복과 슬픔, 고난과 역경 속에서 두 사람은 서로에게 기대며 하나가 되어간다.
이것은 Junkuk Kim 과 Bilqees Kim 의 실제 이야기다.

> "우리의 이야기가 누군가에게 용기가 되기를."
